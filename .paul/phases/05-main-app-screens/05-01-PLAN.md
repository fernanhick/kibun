<objective>
## Goal
Build the HomeScreen with live mood data (today's entries, streak counter, last entry summary) and the HistoryScreen with a monthly calendar view where each day is color-coded by dominant mood. Tapping a day opens a DayDetailScreen showing all entries for that date. After this plan, the user has a meaningful app experience beyond check-in: they can see their mood patterns visually and review past entries.

## Purpose
With Phase 4's check-in loop complete, the app saves entries but offers no way to review them. This plan delivers the two data-driven screens that make logged moods visible and valuable — turning raw data into a visual pattern the user can browse.

- `app/(tabs)/index.tsx` — Enhanced HomeScreen: today's entries list, streak counter, last entry card, Shiba mood-aware greeting
- `app/(tabs)/history.tsx` — HistoryScreen: monthly calendar grid with color-coded day cells, month navigation
- `app/day-detail.tsx` — DayDetailScreen: list of all entries for a selected date

    Add helper functions to moodEntryStore. These are pure functions exposed as store actions for convenience, operating on the entries array.

    ── 1. getEntriesForDate(dateStr: string): MoodEntry[] ─────────────────────

    Filter entries where loggedAt starts with the given YYYY-MM-DD string.
    Return sorted by loggedAt ascending (earliest first — for DayDetailScreen).

    ```ts
    getEntriesForDate: (dateStr: string) => {
      return get().entries
        .filter((e) => e.loggedAt.startsWith(dateStr))
        .sort((a, b) => a.loggedAt.localeCompare(b.loggedAt));
    }
    ```

    ── 2. getDaysWithEntries(yearMonth: string): Record<string, MoodId> ───────

    For a given YYYY-MM prefix, return a map of day strings (YYYY-MM-DD) to the dominant mood ID for that day.
    "Dominant mood" = the mood that appears most frequently that day. On tie, use the most recent entry's mood.

    ```ts
    getDaysWithEntries: (yearMonth: string) => {
      const dayEntries = get().entries.filter((e) => e.loggedAt.startsWith(yearMonth));
      const byDay: Record<string, MoodEntry[]> = {};
      for (const e of dayEntries) {
        const day = e.loggedAt.split('T')[0];
        (byDay[day] ??= []).push(e);
      }
      const result: Record<string, string> = {};
      for (const [day, entries] of Object.entries(byDay)) {
        // Count frequency of each mood
        const freq: Record<string, number> = {};
        for (const e of entries) {
          freq[e.moodId] = (freq[e.moodId] ?? 0) + 1;
        }
        // Find max frequency
        let maxCount = 0;
        for (const count of Object.values(freq)) {
          if (count > maxCount) maxCount = count;
        }
        // Among moods with max frequency, pick the most recent
        const topMoods = Object.keys(freq).filter((m) => freq[m] === maxCount);
        if (topMoods.length === 1) {
          result[day] = topMoods[0];
        } else {
          // Tie-break: most recent entry among tied moods
          const sorted = entries
            .filter((e) => topMoods.includes(e.moodId))
            .sort((a, b) => b.loggedAt.localeCompare(a.loggedAt));
          result[day] = sorted[0].moodId;
        }
      }
      return result;
    }
    ```

    ── 3. getStreak(): number ─────────────────────────────────────────────────

    Count consecutive calendar days (from today backwards) that have at least one entry.
    Today counts if it has entries. If today has no entries, streak = 0.

    ```ts
    getStreak: () => {
      const entries = get().entries;
      if (entries.length === 0) return 0;

      const daysWithEntries = new Set(
        entries.map((e) => e.loggedAt.split('T')[0])
      );

      let streak = 0;
      const d = new Date();
      while (true) {
        const dateStr = d.toISOString().split('T')[0];
        if (!daysWithEntries.has(dateStr)) break;
        streak++;
        d.setDate(d.getDate() - 1);
      }
      return streak;
    }
    ```

    ── 4. Update the MoodEntryState interface ─────────────────────────────────

    ```ts
    interface MoodEntryState {
      entries: MoodEntry[];
      addEntry: (entry: MoodEntry) => void;
      getEntriesForDate: (dateStr: string) => MoodEntry[];
      getDaysWithEntries: (yearMonth: string) => Record<string, string>;
      getStreak: () => number;
    }
    ```

    Note: These methods use `get()` — add the `get` parameter to the persist store creator alongside `set`:
    ```ts
    persist(
      (set, get) => ({ ... })
    ```

    The builder function (defined outside the component or as a module-level function):
    ```ts
    function buildCalendarGrid(year: number, month: number): (number | null)[][] {
      const firstDay = new Date(year, month, 1).getDay(); // 0=Sun
      const daysInMonth = new Date(year, month + 1, 0).getDate();
      const grid: (number | null)[][] = [];
      let week: (number | null)[] = new Array(firstDay).fill(null);

      for (let day = 1; day <= daysInMonth; day++) {
        week.push(day);
        if (week.length === 7) {
          grid.push(week);
          week = [];
        }
      }
      if (week.length > 0) {
        while (week.length < 7) week.push(null);
        grid.push(week);
      }
      return grid;
    }
    ```
